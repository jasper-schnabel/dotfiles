# dotfiles

A simple and powerful archlinux setup, always kinda WIP!

## Features

- zsh
- [neovim](https://github.com/jasper-schnabel/neovim-lua)
- alacritty
- mpd, ncmpcpp and beets
- zathura, nsxiv and mpv
- conky and dunst
- rofi
- sxhkd
- qtile

### Requirements

- archlinux
- neovim nightly
